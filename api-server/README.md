# api-server


## Pre-requisites

* Python version defined in [.python-version](`./.python-version`)


## Install

```sh
# Create a venv
<python> -m venv .venv

# Use the venv
source .venv/bin/activate

# Install the dependencies 
<python> -m pip install -r requirements.txt
```


## Launch Instructions

```sh
litestar run --port 8123
```
